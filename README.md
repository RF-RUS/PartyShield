# PartyShield - VPN Service Website

Современный многостраничный сайт для VPN-сервиса PartyShield с поддержкой русского и английского языков.

## Технологии

- **Next.js 14** - React фреймворк с App Router
- **TypeScript** - типизация
- **Tailwind CSS** - стилизация
- **Framer Motion** - анимации
- **Lucide React** - иконки

## Структура проекта

```
PartyShield/
├── app/
│   ├── page.tsx          # Главная страница
│   ├── pricing/
│   │   └── page.tsx      # Страница с тарифами
│   ├── try/
│   │   └── page.tsx      # Страница "Попробовать"
│   ├── support/
│   │   └── page.tsx      # Страница поддержки
│   ├── layout.tsx        # Корневой layout
│   └── globals.css       # Глобальные стили
├── components/
│   ├── Header.tsx         # Шапка сайта
│   └── Footer.tsx        # Подвал сайта
├── contexts/
│   └── LanguageContext.tsx # Контекст для переключения языков
├── lib/
│   └── i18n.ts           # Переводы (русский/английский)
└── package.json
```

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите dev-сервер:
```bash
npm run dev
```

3. Откройте [http://localhost:3000](http://localhost:3000)

## Сборка для продакшена

```bash
npm run build
npm start
```

## Страницы

### Главная (`/`)
- Hero-секция с CTA кнопками
- Преимущества сервиса
- Отзывы пользователей
- Поддерживаемые устройства

### Цены (`/pricing`)
- 4 тарифных плана (Базовый, Стандарт, Премиум, Бизнес)
- Переключение валют (USD/RUB)
- Кнопки покупки ведут на Telegram-бота

### Попробовать (`/try`)
- Инструкция в 3 шага
- Кнопка открытия Telegram-бота
- Простое описание процесса

### Поддержка (`/support`)
- Ссылка на Telegram-группу поддержки
- Видеоинструкции (заглушки для YouTube/Rumble)
- FAQ с сворачиваемыми блоками
